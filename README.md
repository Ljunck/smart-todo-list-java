# Smart To-Do List (Java)

##ADITYA MISHRA (24BAI10974)

## Problem
Students often struggle to manage tasks, deadlines, and priorities effectively, leading to poor time management and missed deadlines.

## Solution
This project is a CLI-based Smart To-Do List application that helps users manage their tasks efficiently using priority levels and a built-in timer system.

## Features
- Add tasks with priority and deadline
- Built-in timer system
- Track remaining time
- Mark tasks as completed
- Delete tasks
- View pending tasks

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList

## How to Run
javac src/*.java  
java -cp src Main

## Future Improvements
- File storage (save tasks permanently)
- GUI version using Java Swing
